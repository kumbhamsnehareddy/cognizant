Community Portal Help Guide

Getting Started

Navigation

The portal has several main sections:

  • Home: View upcoming events and announcements
  • Events: Browse and register for community events
  • Gallery: View photos from past events
  • Contact: Get in touch with the community organizers

Event Registration

To register for an event:

  1. Click on the event card or "Register Now" button
  2. Fill out the registration form with your details
  3. Review your information and submit
  4. You'll receive a confirmation message

Finding Events Near You

The portal can help you find events in your area:

  1. Click the "Find Nearby Events" button
  2. Allow location access when prompted
  3. View events sorted by distance from your location

Photo Gallery

To view event photos:

  • Click on any image to enlarge it
  • Double-click to return to normal size
  • Hover over images for event details
